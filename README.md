#アニメグッズレビューサイト

サイト概要

サイトテーマ
架空アニメのグッズのレビューサイト​

テーマを選んだ理由
私も何かを購入する際にはレビューサイトの感想等を参考に、可能な限り失敗しないように心掛けております。
特にアニメグッズは世界的な需要も高く、国内外多くの方に閲覧して頂ける可能性が高いと思い、このようなポートフォリオを制作してみようと思いました。

ターゲットユーザ
・購入する商品をお悩みの人
・購入したことのあるユーザーの意見を聞きたい人

主な利用シーン
グッズを購入したいが予算に限りがあり、購入するものしないものに悩んでいる際や同じようなものがあり、どちらを購入するな悩んでいる時に利用できるサービスを想定しました。
